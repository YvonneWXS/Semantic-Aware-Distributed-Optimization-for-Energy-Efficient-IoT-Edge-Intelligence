# Semantic-Aware-Distributed-Optimization-for-Energy-Efficient-IoT-Edge-Intelligence
将语义通信（SemCom）与边缘智能相结合，为减少数据冗余和增强信息传输中的任务相关性提供了一种有前景的范式。然而，动态环境、异构任务要求和严格的延迟约束使得平衡语义保真度和能源效率变得具有挑战性，使卸载和资源分配决策变得复杂。为了解决这些问题，本文提出了一种用于多用户物联网边缘系统的语义感知分布式优化框架。该框架联合优化了任务卸载、语义提取因子、带宽分配和边缘计算资源分配，同时考虑了用户级任务多样性和资源限制。该问题被建模为部分可观测马尔可夫决策过程（POMDP），并使用基于独立近邻策略优化（IPPO）的分布式强化学习算法求解，使用户设备能够从本地观测中学习自适应策略。此外，引入了一种具有自适应惩罚函数的全局奖励机制，以指导违反约束的长期优化。
